# 🛠 클라이언트 개발자의 인프라 엔지니어 도전기

> 프론트엔드 개발자 → 인프라 엔지니어 전환을 위한 학습 기록

프론트엔드 개발 경력을 기반으로 인프라 엔지니어로 전환하기 위한 학습 과정을 기록하는 저장소입니다.  
학습 내용, 실습 코드, 삽질 기록, 자격증 준비 과정을 남깁니다.

---

## 목표

- [ ] AWS Solutions Architect Associate (SAA-C03) 취득
- [ ] HashiCorp Terraform Associate (003) 취득
- [ ] Certified Kubernetes Administrator (CKA) 취득

| Phase   | 내용                         | 기간  | 상태      |
| ------- | ---------------------------- | ----- | --------- |
| Phase 1 | 리눅스 + 네트워크            | ~10일 | 🔄 진행중 |
| Phase 2 | Docker                       | ~6일  | ⬜ 대기   |
| Phase 3 | AWS + SAA 자격증             | ~6주  | ⬜ 대기   |
| Phase 4 | Python for Infra             | ~5일  | ⬜ 대기   |

| Phase 6 | Kubernetes + CKA             | ~5주  | ⬜ 대기   |
| Phase 7 | 모니터링 + 보안 + 트러블슈팅 | ~2주  | ⬜ 대기   |
